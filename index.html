311-Call

This civic project uses data from Los Angeles' 311 calls on non-emergency issues. The goal of the team is to bridge the data into a usable product for neighborhood councils around LA to make decisions. I worked on data cleaning and visualization using Python/pandas & postgreSQL, and with react and javascript on designing a webpage.

Tools & Tech: Python (+pandas, numpy), Jupyter notebook, Sanic, ReactJS, Docker, postgreSQL, SQLAlchemy
Github link

SETI Internship: Detecting and Processing Radio Frequency Interference

I worked with hardware to track and process local cell signals and radio transmissions, and used the Green Bank Telescope for atmospheric noise. The ETL operations are followed by data cleaup, analysis, and visualization. My work is saved in Jupyter notebooks with instructional documentation as reference material for future interns and students.

Tools: Python/numpy, hackrf, Jupyter notebook, GNU Radio
Jupyter notebook: Cellphone transmissions
Jupyter notebook: Radio stations

MERN App: Associated Press Article Scraper

A foundational full-stack app, it scrapes information from the Associated Press. Visitors can save the scraped articles into a mongo database and can leave comments under an article. The design is responsive and works nicely on mobile. The app is containerized using Docker and is deployed on Render.com.

Tools: Docker, MongoDB, Express, javascript, React.js, node.js, HTML/CSS
Github Link

Solar System Simulator

Inspired by the idea of accurately depicting the barycenter of the Solar System as being outside of the Sun, we crafted this project primarily using HTML and CSS. I used available open-source code to re-create a live planet/Moon tracker based on user location (longitude/latitude).

Tools: HTML, CSS, NASA API, math
